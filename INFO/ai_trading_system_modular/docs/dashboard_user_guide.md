# Tradecraft Dashboard — User Guide (Condensed)

## What It Does
- Place market/pending orders with **risk-aware auto‑fill** for SL/TP using PIPS/ATR/%.
- Show balances, equity, exposure, recent deals, symbol analysis.
- Persist risk settings: Default %, Max %, target R:R.

## How To Use
1. Open Dashboard → **Order Entry** tab.
2. Select Symbol + Volume → choose strategy (PIPS/ATR/%).
3. Let auto‑fill compute SL/TP; adjust if needed.
4. Place Market or Pending order.
5. Monitor **Analysis** page for performance & exposure.
