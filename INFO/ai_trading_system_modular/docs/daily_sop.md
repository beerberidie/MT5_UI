# Daily Operating SOP

**Morning (09:00 SAST)**
1. Run data generator; validate JSON; save CSV rows.
2. Review macro themes (DXY, US10Y, VIX) and top headlines.

**Trade Window (09:30–10:00 SAST)**
3. Run Perplexity daily trade prompt; ensure ≥1:2 RR.
4. Calculate lot size for 2–5% risk; confirm SL/TP vs news calendar.
5. Execute via MT5 or Dashboard.

**Post-Trade (Evening)**
6. Log results in trade journal; screenshot chart; write brief notes.
7. Review mistakes; update next‑day watchlist.
